### كيفية تسجيل الدخول كمدير:
1. اضغط على أيقونة القفل (🔒) في الصفحة الرئيسية
2. أدخل كود الإدارة الخاص بك
3. ستحصل على رسالة ترحيب شخصية باسمك
4. ستظهر لك لوحة التحكم للإدارة

### صلاحيات المديرين:
- إضافة وتعديل وحذف الألعاب
- إدارة التصنيفات (إضافة، تعديل، حذف)
- عرض معلومات من أضاف/عدّل كل عنصر
- الوصول الكامل لجميع ميزات الإدارة

### الميزات:

- ✅ عرض الألعاب مع الصور والفيديوهات
- ✅ البحث في الألعاب
- ✅ تصنيف الألعاب حسب الفئات
- ✅ لوحة تحكم للإدارة مع أسماء المديرين
- ✅ تتبع من أضاف/عدّل كل عنصر
- ✅ دعم كامل للغة العربية
- ✅ تصميم متجاوب لجميع الأجهزة
- ✅ نظام مصادقة آمن للمديرين

### التقنيات المستخدمة:

- React 19
- TypeScript
- Tailwind CSS
- Convex (قاعدة البيانات والمصادقة)
- Vite (أداة البناء)
